# 📖 About EduMate

EduMate is a comprehensive College Management System developed using Flutter that streamlines academic activities for educational institutions.

The application provides dedicated dashboards for students, teachers, and administrators, enabling efficient management of attendance, study materials, notices, examination results, academic records, and AI-powered student assistance.

The goal of EduMate is to reduce paperwork, improve communication, and provide a smart digital platform for colleges.

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Rohanmon80/edumate.git
```

Navigate into the project

```bash
cd edumate
```

Install dependencies

```bash
flutter pub get
```

Run the application

```bash
flutter run
```

---

# 📦 Build Release APK

```bash
flutter build apk --release
```

Generated APK:

```
build/app/outputs/flutter-apk/app-release.apk
```
